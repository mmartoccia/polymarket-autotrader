# Polymarket AutoTrader Dashboards

## Unified Dashboard (Recommended)

The **unified dashboard** combines all monitoring features into a single view:

### Usage

From project root:
```bash
./dashboard.sh
```

Or directly:
```bash
python3 dashboard/unified_dashboard.py
```

### Features

- **Balance & Status**: Current balance, mode, peak, drawdown, daily P&L
- **Epoch Timing**: Progress bar showing time in current 15-minute epoch
- **Open Positions**: Active positions with P&L, losing positions summary
- **Agent Decisions**: Recent approvals, vetoes, and skipped trades
- **Auto-refresh**: Updates every 5 seconds
- **Color-coded**: Green for gains, red for losses, yellow for warnings

## Legacy Dashboards

### Live Dashboard (Python)
```bash
python3 dashboard/live_dashboard.py
```
- Detailed position tracking
- Market details
- 10-second refresh

### Agent Dashboard (Bash)
```bash
/tmp/agent_dashboard_final.sh
```
- Simple agent vote display
- 5-second refresh
- Lightweight SSH-based

## Requirements

- SSH access to VPS configured (`~/.ssh/polymarket_vultr`)
- Python 3.8+
- `requests` library

## Configuration

Edit the constants in `unified_dashboard.py` if needed:

```python
VPS_IP = "216.238.85.11"
SSH_KEY = "~/.ssh/polymarket_vultr"
WALLET = "0x52dF6Dc5DE31DD844d9E432A0821BC86924C2237"
REFRESH_INTERVAL = 5  # seconds
```
